# LLM Personality Agent

A Python project exploring personality-driven prompt engineering with a local large language model. The agent is configured with a defined persona and responds to user input in a consistent, character-consistent voice.

## Purpose
Demonstrates how system-level prompt design and LLM configuration parameters can shape AI agent behavior and personality — a foundational concept in AI agent development.

## Tech Stack
Python · llama-cpp-python · Prompt Engineering · LLM Configuration

## How to Run
1. Install dependencies
2. Run: `python main.py`
